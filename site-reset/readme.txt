=== AutoBlogged Site Reset ===
Tags: AutoBlogged, Delete, Reset, Remove, Empty, Delete Posts
Requires at least: 2.8
Tested up to: 3.1

Resets your blog to it's initial state, removing all posts, comments, categories, and tags.

== Description ==
This plugin creates a Reset Site option under the tools menu that esets your blog to it's initial state, removing all posts, comments, categories, tags, and emptying the trash. This is useful while testing a new AutoBlogged installation, allowing you to reset the site back to it's original state.

Note that this plugin will not remove any custom post types you may have in your blog.

== Installation ==

Upload and activate the plugin through by using the Add New button on the WordPress Plugins admin page. To manually install the plugin, create a new directory called site-reset in the /wp-content/plugins directory of your WordPress installation. Extract the zip file and upload the contents to the /wp-content/plugins/site-reset directory and then activate the Plugin from Plugins page.
